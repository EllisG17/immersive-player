# Features:

## 1. Layout Editor:
    This can be accessed via the 'settings' in the top right of my extension.
   
![Alt text](https://i.imgur.com/x838YEQ.png)

All features are expandable via the bottom right corner. There is a snap grid option to allow precise placement, 'still in beta' All features can be hidden via the settings and moved by dragging the top bar.

## 2. Vinyl Look:
    This feature makes the album cover of the song playing become a circle and spin like an album in a record player. There is an option to enable the 'tonearm' for aesthetics.

![Alt text](https://i.imgur.com/cb7Q50j.png)

## 3. Tonearm:
   This feature allows for the album cover to have a Tonearm overlay in vinyl mode. There are 4 Different options to choose from. ranging from 'Classic', 'Minimal Straight', 'S-curve', and 'Retro brass'
